# Simulation Model

## Model inventory

Document the robot, end effector, D-sub 9 connector, receptacle, fixtures, and collision geometry, including source, license, scale, and version.

## Coordinate frames

Define the world, robot base, tool center point, connector, and receptacle frames. Record axis directions, units, handedness, and the transform convention used in code and logs.

## Contact and dynamics

Record mass, inertia, friction, compliance, damping, solver settings, and simulation timestep. Justify parameters that materially affect insertion forces or stability.

## Validation

Check model scale, joint limits, collision pairs, transform round trips, static stability, and representative contact behavior before running experiments.
